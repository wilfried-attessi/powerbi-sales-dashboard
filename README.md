# Sales Performance Dashboard (Power BI)

Interactive 2-page Power BI dashboard analyzing retail sales and profitability
(~$50M in sales, 50K orders).

## Pages
- **Revenue Focus** — total sales by product, profit-contribution waterfall, margin gauge, drill-down by year/category/segment/region.
- **Profitability Focus** — profit margin by segment, sales by category & segment, orders by quarter, region × category matrix.

## Tools
Power BI · DAX (measures, star schema) · interactive time slicers (Day / Month / Year).

## Files
- `Wilfried_Attessi_Assn1.pbix` — Power BI report
- `dashboard_revenue.png`, `dashboard_profitability.png` — screenshots

> Note: open the `.pbix` with Power BI Desktop. Screenshots are provided for quick preview.
